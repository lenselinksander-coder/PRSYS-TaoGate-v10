3️⃣ Conflictmechanisme

Regels kunnen botsen.

Bijvoorbeeld:
	•	EU zegt: BLOCK
	•	Gemeente zegt: toegestaan

Olympia-algoritme:

def resolve_conflict(rules):
    sorted_rules = sort_by_priority(rules)
    for rule in sorted_rules:
        if rule["action"] == "BLOCK":
            return "BLOCK"
    return highest_priority_decision(rules)

BLOCK wint altijd.
Hogere jurisdictie wint bij conflict.

Dat is mechanisch, niet politiek.

6️⃣ Concrete minimal viable Olympia

Voor jouw Replit-agent:

def olympia_execute(scope_result, rule_registry):
    applicable = [
        r for r in rule_registry
        if r["domain"] == "AI"
        and r["scope"] == scope_result["scope"]
    ]

    applicable_sorted = sorted(
        applicable,
        key=lambda r: PRIORITY_ORDER.index(r["layer"])
    )

    for rule in applicable_sorted:
        if rule["action"] == "BLOCK":
            return rule

    return applicable_sorted[0] if applicable_sorted else None
